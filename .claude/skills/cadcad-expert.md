N_PATHS = 1000
RANDOM_SEED = 42
np.random.seed(RANDOM_SEED)

# Log-normal demand shocks
def demand_shock(t, scenario_params):
    mu = scenario_params["demand_growth"] / 12  # monthly
    sigma = 0.60 / np.sqrt(12)                  # 60% annual vol, monthly
    return np.random.lognormal(mu - 0.5 * sigma**2, sigma)

## Reflexivity Loop (required in staking.py)

```python
def staking_step(state, params):
    APY_t = compute_APY(state['staking_rate'], state['emission_rate'], state['price'])
    opportunity_cost = params['opportunity_cost']  # default 0.05
    sensitivity = params['staking_sensitivity']    # default +1% staking per +0.5% APY above OC
    
    delta_staking = sensitivity * (APY_t - opportunity_cost)
    
    # Mass unstaking trigger
    if state['price_30d_change'] < -0.40:
        delta_staking -= 0.20  # 20% of staked supply unstakes
    
    new_staking_rate = np.clip(state['staking_rate'] + delta_staking, 0, 1)
    new_circulating = state['total_supply'] - state['total_supply'] * new_staking_rate
    
    return new_staking_rate, new_circulating
```

## Error Handling

Scripts must not crash silently. If a required parameter is missing:
```python
required = ['initial_circulating', 'total_supply', 'emission_rate_annual_pct']
for param in required:
    if param not in PARAMS or PARAMS[param] == 'unknown':
        raise ValueError(f"Required parameter '{param}' is unknown. Cannot run simulation.")
```

If a parameter is unknown but not required (simulation can run with a default), log the default being used.

## Dilution Event Detection

Flag months where new supply > 5% of circulating:
```python
df['dilution_event'] = (df['new_supply_pct'] > 0.05)
dilution_months = df[df['dilution_event']].index.tolist()
if dilution_months:
    print(f"WARNING: Dilution events detected at months: {dilution_months}")
```
Mark these months with red vertical lines on the supply chart.
